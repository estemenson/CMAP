h1. Welcome to CMAP

This is the oficial source repository for the Collaborative Multi-touch Agile Planner (CMAP). This is a research project and information about objectives and results can be found on our wiki.

You will find the project documentation on the "wiki":http:git@github.com/estemenson/CMAP.wiki.

Issues and feature suggestions can be created in our "issue tracker":http://github.com/estemenson/CMAP/issues.
